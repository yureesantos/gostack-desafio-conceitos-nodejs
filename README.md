## :computer: Tech:

- NodeJS
- Express
- ESLint + Prettier

## :running: Run the project:

```shell
  # Clone this repository
  - git clone https://github.com/izabela-am/GoStack-Desafio-Conceitos-NodeJS.git

  # Enter project directory
  - cd [project folder]

  # Install dependencies
  - yarn

  # Run the tests:
  - yarn test

  # Start the server
  - yarn dev
```
